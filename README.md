# OffensiveAgentic

The open reference for the **offensive security of Agentic AI** — red teaming, penetration testing, vulnerability assessment, and evaluation/benchmarking of autonomous AI agents.

A minimalist, **no-build** static site. Home page = category cards + a hierarchy tree of the whole field; each category opens its own page with search and filtering. Add a resource by pasting a short block into `data.js`. See **[HOW-TO-EDIT.md](HOW-TO-EDIT.md)**.

## Files
`index.html` (home) · `category.html` (per-category template) · `config.js` (categories + tree) · `data.js` (content) · `app.js` (logic) · `style.css` (design)

## Preview
Double-click `index.html`.

## Publish on GitHub Pages
**Settings → Pages → Source → Deploy from a branch → `main` / root.** Live at
`https://apowfour.github.io/Offensive-Security-of-Agentic-AI/`. No Actions workflow needed.
