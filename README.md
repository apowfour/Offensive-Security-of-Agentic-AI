# OffensiveAgentic

The open reference for the **offensive security of Agentic AI** — red teaming, penetration testing, vulnerability assessment, and evaluation/benchmarking of autonomous AI agents.

A minimalist, single-page static site. **No build step** — three files (`index.html`, `style.css`, `data.js`). Add a resource by pasting a short block into `data.js`. See **[HOW-TO-EDIT.md](HOW-TO-EDIT.md)**.

## Preview
Double-click `index.html`. (Fonts and the mind map load from a CDN, so an internet connection shows them at their best.)

## Publish on GitHub Pages
1. Put these files in your repo.
2. **Settings → Pages → Source → Deploy from a branch → `main` / root**.
3. Live at `https://apowfour.github.io/Offensive-Security-of-Agentic-AI/`.

Because it's plain static files, no GitHub Actions workflow is required — Pages serves them directly.
